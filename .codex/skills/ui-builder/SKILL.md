# UI Builder Skill

Use this skill when asked to create or improve UI.

## Read first
- AGENTS.md
- .codex/memory/product.md
- .codex/memory/roadmap.md

## UI principles
- Dark dashboard
- Blue/cyan accent
- Clean spacing
- Responsive from mobile to desktop
- No clutter
- Use clear empty states
- Make buttons obvious

## Components to prefer
- AppShell
- Sidebar
- Dashboard
- InterestsManager
- TopicCard
- TopicFinder
- PostCreator
- LinkedInPreview
- ImageCard
- DraftsList

## Rules
- Do not add backend unless requested.
- Do not add Firebase.
- Use mock data when backend is not ready.
- Keep components typed.
- Run build after changes.
