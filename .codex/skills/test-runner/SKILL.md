# Test Runner Skill

Use this skill when build, lint, test, or runtime checks fail.

## Commands
Start with:
npm run build

If available:
npm run lint
npm test

## Rules
- Do not hide errors.
- Fix root causes.
- If dependency missing, explain and install only if necessary.
- Keep changes focused.
