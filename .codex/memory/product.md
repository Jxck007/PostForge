# Product Memory: PostForge

## Owner
Jack, CSE student and builder.

## Product purpose
PostForge helps Jack stay active on LinkedIn without manually thinking of posts every time.

It should create:
- LinkedIn caption
- hashtags
- image/card title
- image/card subtitle
- final ready-to-post preview

## User persona
Student developer interested in:
- AI
- Python
- Linux
- Cloud
- React
- Firebase
- DSA
- Hackathons
- Local LLMs
- Student projects

## Desired output
Human-in-the-loop LinkedIn content preparation.
The user must manually post on LinkedIn.
